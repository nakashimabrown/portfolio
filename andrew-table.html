Splay Legged Table
Solo project
An exploration of contemporary Ming furniture
Cut entirely using a small hobbyist's tablesaw
Designed and built by
Hugo Nakashima-Brown
This table was the first piece inspired by Ming Furniture that I designed and built, increasing splay and cantilever and using a slab top, as opposed to the traditional frame and panel. It was built this was to suggest the timelessness of Ming design with a few tweaks for more contemporary taste, while still using traditonal joinery but only cut by machines available in any garage shop. I recently rebuilt this table with Fine Woodworking over a three day shoot to popularize making such joinery, which is usually viewed through a mystified lens as unapproachable, to become another, non-Eurocentric part of makers' toolkits.
The joinery was cut using a simple tablesaw jig with a flat top blade, giving zero clearance cuts to line up your parts and set stops if preferred, using the tablesaw almost like traditional hand tools. Angles of any degree could be easily screwed on as needed.

Leading to a joint with perfect clearances that is tight when assembled but easily disassembleable for moving and shipping.
